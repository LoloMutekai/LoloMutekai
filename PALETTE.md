# Palette — "Encre & rose fané"

> Palette signature du profil, faite main, fixe, indépendante de matugen.
> Esthétique : 床の間 / yūgen — sombre profond, froid sourd, **percé d'un seul rose fané** (le "tell").
> Le décalage femboy/PhD+ se dit ici, indirectement : un rose triste qui perce l'austérité.

| Rôle | Hex | Usage |
|---|---|---|
| `--ink`     | `#0d0e12` | Base / fond profond (encre) |
| `--surface` | `#16131a` | Surfaces, cartes (charbon tiédi) |
| `--text`    | `#8a93a6` | Texte principal (gris-bleu sourd) |
| `--muted`   | `#5a6070` | Texte secondaire, traits (gris ardoise) |
| `--tell`    | `#c98ba0` | **Accent unique** — rose fané poudré (le "tell", à doser) |
| `--tell-dim`| `#9c6f80` | Rose assombri (états atténués) |
| `--line`    | `#2a2730` | Filets, séparateurs discrets |

**Règle d'or (council)** : tout élément du profil — y compris widgets tiers — adopte cette palette,
sinon il dégage. Densité monochromatiquement disciplinée = le yūgen survit à l'empilement.

**Réutilisation** : cette palette est destinée à devenir la charte graphique Kairo (bannière repo
vitrine, headers READMEs, splash kairo-dashboard, infusion future matugen).
